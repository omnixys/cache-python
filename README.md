# omnixys-cache

Omnixys shared cache package with Redis/Valkey client, serialization, and invalidation.

## Installation

```bash
pip install omnixys-cache
```

## Features

- Redis/Valkey client wrapper
- JSON serialization for cache values
- Cache invalidation strategies
- Type-safe cache key definitions

## Usage

```python
from omnixys_cache import CacheClient, CacheInvalidationService, JsonCacheSerializer
```
